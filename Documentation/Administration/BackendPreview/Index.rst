Backend Preview
===============

This extension ships the possibility to add a backend preview, which shows the
typed categories in an information box on the backend page.

..  figure:: /Images/Administration/BackendPreview/backend-preview-box.png
    :alt: Screenshot of the backend preview info box
    :width: 400px

    The backend preview in the page view

For activating this feature, just install the extension `page_backend_layout`_.

.. _page_backend_layout: https://extensions.typo3.org/extension/page_backend_layout
